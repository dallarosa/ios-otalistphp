#iOS OTA List PHP

Auto-generates a list of the `.plist` for the  `.ipa` files on the current directory.

## Usage
1. Just copy all the files where your `.plist` files are.
2. Rename `htaccess.txt` to `.htaccess`.
3. Run trough php and the browser. Done!
